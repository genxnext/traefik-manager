    Export Configuration

    Download your Traefik configuration or create a full backup for migration

    Full Backup

    Exports all etcd keys under /traefik/ plus your connection profiles (no user accounts). Use this to fully replicate a setup on a new install.

    • All HTTP/TCP/UDP/TLS router, service, middleware keys
    • TLS options, stores, domains
    • etcd connection profiles (URL + name)
    • Not included: user accounts & passwords
    Download Full Backup .json

    Config Snapshot

    Exports parsed routers, services, and middlewares as a structured JSON snapshot. Useful for version-control or sharing individual config blocks.

    • Parsed HTTP routers, services, middlewares
    • Structured format (importable via Import page)
    • Does not contain raw etcd keys or connections
    •  
    Download Config Snapshot .json

    Restore a backup

    To restore a full backup on a fresh install, go to Import → Full Backup and upload the .json file you downloaded here.
