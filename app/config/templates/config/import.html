    Upload a Full Backup file (.json) previously downloaded from Export. This restores all etcd key-value pairs and, optionally, connection profiles.

    This will overwrite existing etcd keys. All Traefik routes, services, middlewares and TLS settings stored in etcd will be replaced with the values from the backup file. This action cannot be undone.
    Select a .json file exported via Export → Full Backup.
    {{ form_footer(url_for('config.export_config'), 'Restore Backup') }}

    BACKUP FILE STRUCTURE

    {
  "version": "1.0",
  "exported_at": "2025-01-15T10:30:00Z",
  "etcd_prefix": "/traefik/",
  "etcd_kvs": {
    "/traefik/http/routers/my-router/rule": "Host(`example.com`)",
    ...
  },
  "connections": [
    {"name": "prod", "url": "http://etcd:2379", "description": "..."}
  ]
}
